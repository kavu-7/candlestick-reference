# 📊 The Trading Desk — Candlestick Pattern Reference

An interactive dark-themed flashcard reference for 22 candlestick patterns.
Built for paper trading study and quick market reference.

## 🔗 [Open Live Reference → https://kavu-7.github.io/candlestick-reference/

## Features
- 22 patterns — single & multi candle, fully categorised
- Flip cards with conditions, entry, stop loss & targets
- Live OHLC calculator — enter real prices, get your exact SL & 4 targets
- Filter by Bullish / Bearish / Neutral
- Sort by reliability %
- Risk Taker vs Risk Averse guidance for every pattern
- Fully offline — single HTML file, no dependencies

## Important
Patterns are entry signals, not complete systems.
Always combine with: Support & Resistance · Volume · RSI/MACD · Trend Structure
